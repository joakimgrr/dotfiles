# dotfiles
Very much WIP

# Usage
1. Git clone this to ~/dotfiles
2. Run: ./bootsrap.sh [osx / ubuntu] 

Running "./bootsrap.sh [environment]" will create backups from existing dotfiles and replace them with symlinks to these ones.
